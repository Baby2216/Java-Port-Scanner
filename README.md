# Java Port Scanner

## Description
A Java-based network port scanner that identifies open ports on a target system.

## Features
- Scan ports 1-1024
- Detect open ports
- Display results in console
- Lightweight and easy to use

## Technologies Used
- Java
- Socket Programming
- Networking

## How to Run

1. Compile the program:

```bash
javac PortScanner.java
```

2. Run the program:

```bash
java PortScanner
```

## Sample Output

Scanning host: 127.0.0.1

Port 80 is OPEN
Port 443 is OPEN

Scan Complete.
